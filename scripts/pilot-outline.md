# Pilot Episode Outline

Main topics covered:

1. **many/much** - Using "beaucoup de" for both countable and uncountable nouns.
2. **do/does vs. is/are** - Simple questions with do/does and is/are.
3. **vowel pronunciation** - Pronunciation of French vowels, especially "u" vs "ou".
4. **false friends** - Common false friends between English and French.
